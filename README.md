
# Personal Portfolio Website

## Overview
This project consists of creating a personal portfolio website that showcases my skills, projects, and professional identity as a web developer. The purpose of this assignment is to apply HTML, CSS, and JavaScript fundamentals while building a real-world project that can be presented to potential employers or clients.

---

## Learning Objectives
- Implement modern CSS layouts (Flexbox and/or Grid)
- Create an intuitive and accessible user interface
- Develop a cohesive visual design with consistent branding
